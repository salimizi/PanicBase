# PanicBase

**Analyseur de panic logs iPhone pour techniciens de réparation — Windows**

PanicBase lit les panic logs d'un iPhone (via USB ou import de fichier `.ips` / `.panic`), identifie les pannes matérielles et génère des rapports d'aide à la réparation. Conçu pour les ateliers indépendants et les techniciens avancés.

---

## Fonctionnalités

- **Analyse de panic logs** — diagnostic avec score de confiance, cause probable et vérifications recommandées
- **Détection USB iPhone** — connexion/déconnexion en temps réel, sans iTunes
- **Extraction directe** — récupère les panic logs depuis l'iPhone connecté en USB
- **Import de fichiers** — glisser-déposer ou sélection de fichiers `.ips` / `.panic`
- **Transfert de galerie** — parcourir, prévisualiser et exporter photos/vidéos depuis l'iPhone en USB
- **Infos appareil** — lecture des informations complètes du téléphone en mode normal
- **Mode recovery** — récupération du numéro de série en mode recovery
- **iCloud Photos** — parcourir et télécharger la photothèque iCloud
- **Base de données locale chiffrée** — toutes les données stockées sont chiffrées
- **Anonymisation** — IMEI, UDID, Numéro de série retirés avant stockage
- **Interface multilingue** — Français, Anglais

---

## Installation

Télécharger le dernier installeur depuis la page **[Releases](../../releases)** :

- `PanicBase_x64-setup.exe` — installeur NSIS (recommandé)
- `PanicBase_x64_en-US.msi` — package MSI

**Système requis :** Windows 10 / 11 (64-bit)

Pour la détection USB iPhone, Apple Mobile Device Support doit être installé (inclus avec iTunes ou l'application Apple Devices du Microsoft Store).
